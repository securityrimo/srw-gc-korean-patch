# 슈퍼로봇대전 GC 한글패치 v1.0.0

일본판 `GRWJD9` 전용 첫 공개 릴리스입니다.

## 주요 내용

- 대사와 메뉴 한국어화
- 일본판 이벤트·화자 순서 기준으로 데이터 재구성
- 한국어 글꼴과 런타임 코드북 적용
- 영문판 패치의 타이틀·배너·크레디트 흔적 제거
- 남성 주인공 이름을 `아카츠키 아키미`로 통일
- 여성 주인공 이름을 `아카츠키 아케미`로 통일

## 적용

릴리스의 `SRW_GC_Korean_v1.0.0.xdelta`, `APPLY_PATCH.bat`, `apply_patch.ps1`를 같은 폴더에 두고, 별도로 준비한 `xdelta3.exe`와 함께 사용하세요. 자세한 절차는 저장소의 README를 확인하세요.
