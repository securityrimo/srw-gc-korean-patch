# 슈퍼로봇대전 GC 한글패치 v1.0.5

v1.0.4에서 정신기 설명 23건이 다른 정신기에 연결된 문제와 일부 축약·중복 표기를 수정한 릴리스입니다. 인물·로봇도감 런타임 수정, PDF 번역 품질 교정, 전투 유닛 그래픽 보호 방식과 일본판 이벤트 순서는 그대로 유지합니다.

## 정신기 표기 및 설명 수정

- 정신기 전체 명칭 30건, 축약 30건, 설명 29건을 일본판의 고정 ID와 일대일로 재검증
- 번뜩임 설명에 저격 효과가 표시되는 등 `r0003~r0025` 설명 23건이 연쇄 이동한 문제 수정
- 이미지처럼 표시되는 1글자 축약의 `자기→자`, `력→노`, `난→란` 오류 수정
- 실행 파일에 중복 저장된 `근성 도 근성→근성　초근성`, `카쿠란→교란`, `힘→기력`, `애→사랑` 표기 수정
- 보이는 설명 29건 전부를 최대 2줄·한 줄 23자·41바이트 이내로 정리하고 숨김 `？？？` 슬롯은 일본판 그대로 보존
- 원인이었던 정신기 설명 CSV 순번 매칭 결과를 안정 ID 교정으로 덮어쓰고 이름·축약·설명의 전체 배열을 검사하는 회귀 방지 도구 추가
- 정신기 관련 공통 UI 이미지 14건은 승인본과 픽셀 단위 일치, 필요한 한글 글리프 누락 0건 확인

## 인물·로봇도감 런타임 수정

- 인물도감 267건과 로봇도감 238건, 총 505건의 설명문을 일본판 도감 렌더러 규격에 맞춰 재배치
- 각 줄을 최대 24열·48바이트로 제한하고 ASCII 문자·숫자·공백을 2바이트 표시 셀로 정규화
- 줄바꿈 표식의 홀수 바이트 오프셋, 홀수 바이트 구간, 잘못된 16비트 문자 단위, 헤더 줄 수 불일치 및 끝 표식 누락 0건
- 문제 빌드 회귀 검사에서 2천 건 이상의 구조 오류를 검출하고, 수정 빌드에서는 동일 검사 오류 0건 확인
- 인물도감의 `기도・조타로` 오기를 `키도・죠타로`로 교정

## 번역 품질 교정

- 안정 ID 기준 실제 텍스트 2,778건 교정
- 초반 압사라스 실험장, 올리비 설득 장면 등 대사 95건 직접 교정
- `하야토`, `코우지`, `테츠야`, `아스카`, `진`, `츠토무`처럼 조사·호칭이 붙어 누락됐던 인명 표기 정리
- `모가미 중공`, `소울 세이버`, `다이오쟈`, `바쿠신거`, `수전기대`, `바쿠류오`, `사악수` 등 반복 용어 통일
- 다른 레코드 문장이 섞이거나 내부 placeholder만 제거됐던 도감 장문 13건을 일본어 원문에서 전면 재번역
- 번역 맵의 내부 placeholder와 일본어 문자 잔존 0건

## 시나리오 제목 이미지

- 시나리오 제목 39건을 PDF와 일본어 원문으로 재검토
- 이미 정확했던 2건을 제외한 독립 BMP 37개를 글꼴 제작에 사용한 굵은 맑은 고딕으로 다시 렌더링
- 에피소드 번호↔블록 번호 허용 목록과 원본 블록 SHA-256을 고정해 다른 UI 블록 오염 방지
- PDF 내부 표기가 충돌하는 경우 일본어 원문과 통용 표기를 우선하고, 남녀 주인공 이름은 각각 `아카츠키 아키미`와 `아카츠키 아케미`로 유지

## 적용

수정되지 않은 일본판 원본 ISO에 `SRW_GC_Korean_v1.0.5.xdelta`를 적용하세요. 이전 한글패치가 적용된 ISO에는 덧씌우지 마세요. 릴리스의 `APPLY_PATCH.bat`, `apply_patch.ps1`와 별도로 준비한 `xdelta3.exe`를 같은 폴더에 두고 사용합니다.
